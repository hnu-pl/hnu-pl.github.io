```


    True

```

Often better to give a more speicfic type signature


```haskell
ageOf' :: Text -> Value -> Parser Int
ageOf' name = withObject "addressbook" $ \o -> do
  info <- o .: name
  age <- info .: "age"
  return age
```


```haskell
parseEither (ageOf' "Kate") v
parseEither (ageOf' "Jane") v
```


    Right 15



    Right 19
